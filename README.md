# NOOR Fashion Hub
NOOR Fashion Hub is a modern and responsive fashion e-commerce frontend built with HTML, CSS, and JavaScript. It provides a beautiful shopping experience for customers and includes an admin dashboard for managing store activities.
## ✨ Features
### 🛍️ Customer Storefront
* Responsive and modern UI
* Product categories: Women, Men, Children, Jewelry, Cosmetics, and Shoes
* Product search and sorting
* Shopping cart functionality
* Checkout with multiple payment options
* User Login and Sign Up interface
* User Dashboard with profile and orders
* Interactive chatbot interface

### 📊 Admin Dashboard
* Revenue overview
* Orders analytics
* Customer statistics
* Average order value tracking
* Revenue charts using Chart.js
* Order status visualization
* Recent orders section
* Low-stock product alerts
* Top-selling products section

## 🛠️ Technologies Used
* HTML5
* CSS3
* JavaScript (ES6)
## 🚀 Getting Started
1. Clone the repository
2. Open the project folder
cd noor-fashion-hub
3. Run the website
Open index.html in your browser.
